# vite-vue-template

Vite+Vue项目初始模板，已集成Vuex，VueRouter，Axios等其他主要插件，并且已配置黑夜模式。

>本项目纯作为模板使用，无附加设计。


#### 技术栈

- Vite3
- Vue3 
- Vuex
- Vue Router 
- Axios 
- element-plus 
- vueuse 
- js-cookie 
- normalize.css
- eslint + prettier

    ……

#### 常用指令

- 安装

```
yarn install
```

- 启动

```
yarn dev
```

- 编译

```
yarn build
```

- 检查和修复文件

```
yarn lint
```

- 格式化代码

```
yarn format
```
